SoleMirror

The SoleMirror is a CodeMirror JavaScript Console. Type below and use control-enter to execute the code.

  • Use Control-Up and Control-Down to move through the history.
  • Use Control-Enter to execute the code.
  • Use Control-Space to autocomplete

You can write to the console output using con.log(thingToLog, otherThingToLog...).

To run SoleMirror on any (http) page, bookmark this.
